# 手機端回歸測試清單（手動）

## 基本消息
- 文本發送：正常/空白阻止/Enter 發送/Shift+Enter 換行。
- 圖片：URL 及本地文件（成功顯示、破圖提示）。
- 音頻：URL 播放、播放失敗提示。
- 音樂卡片：播放/暫停/進度，無 URL 時提示。
- 轉帳卡片：確認收款提示。

## 會話/世界書
- 會話：新建/切換/重命名/刪除/清空，草稿與歷史保留。
- 世界書：導入 ST JSON、啟用/切換、導出 JSON、指示器同步。

## 配置與發送
- 配置面板：必填校驗、保存、測試連接；顯示/隱藏 API Key。
- 發送：未配置阻止；離線阻止，恢復後可發送；流式/非流式切換正常。
- 錯誤重試：發送失敗時 Banner「重試」有效。

## 媒體與預覽
- 圖片預覽：點擊放大，關閉正常。
- 音頻/音樂播放：播放/暫停/結束狀態更新，錯誤提示。

## 網絡與狀態
- 切換飛行模式：顯示離線 Banner，發送按鈕禁用；恢復網絡提示並可發送。
- 弱網/失敗：API 返回錯誤時有 toast + Banner 提示。

## 鍵盤與佈局
- 鍵盤彈出時輸入區不被遮擋，焦點時滾動到底。
- 標題/徽章換行不擠壓；列表滾動正常。

## 資產與外觀
- 已下載資產正常顯示；缺失資產不阻斷功能。
